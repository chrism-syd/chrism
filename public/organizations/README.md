# Organization Logos

Place your organization logo files here. The filename should match the `logo_storage_path` value in the organizations table.

Example structure:
- `public/organizations/st-patricks-logo.png`
- `public/organizations/knights-of-columbus-logo.svg`

Database `logo_storage_path` values would be:
- `st-patricks-logo.png`
- `knights-of-columbus-logo.svg`

Supported formats: PNG, JPG, SVG, WebP
Recommended size: 48x48px or 96x96px for crisp display
